# Bank Al-Isra (بنك الإسراء)

منصة أسئلة الامتحانات السابقة للطلاب مع لوحة إدارة كاملة لتجميع وتنظيم بنك الأسئلة.

## نظرة عامة
- **وضع الطلاب:** حل الأسئلة مباشرة بدون تسجيل دخول.
- **وضع الإدارة:** دخول خاص للأدمن لإدارة المواد والأسئلة والمراجعات.
- **تصنيفات الامتحان:** منتصف الفصل / نهائي / كلاهما.
- **مصدر السؤال:** أسئلة امتحانات سابقة أو أسئلة مولدة بالذكاء الاصطناعي.

---

## التشغيل محلياً

1) إنشاء بيئة افتراضية وتثبيت المتطلبات
```
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2) تشغيل الخادم
```
python app.py
```

3) فتح الموقع
```
http://127.0.0.1:8000
```

---

## إعدادات مهمة قبل النشر
- غيّر `SECRET_KEY` في `app.py`.
- غيّر كلمة مرور الأدمن في `app.py`.
- لا تستخدم كلمات مرور حقيقية أثناء التجربة.

---

## المزايا
- تصفّح المواد وعرض الأسئلة حسب نوع الامتحان.
- إجابات فورية مع توضيح الصحيح/الخاطئ.
- شرح يظهر عند الإجابة الخاطئة.
- تنقّل سريع بين السؤال السابق/التالي.
- لوحة إدارة لإضافة وتعديل وحذف الأسئلة.
- اقتراحات الطلاب ورسائل التواصل.
- بلاغات أخطاء مع تصحيح مباشر.
- تصدير واستيراد الأسئلة بصيغة CSV.
- دعم صور الأسئلة + قصّ قبل الرفع.
- تكبير الصور مع زوم داخل العارض.

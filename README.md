# Bank Al-Isra (بنك الإسراء)

## تشغيل المشروع

1) إنشاء بيئة افتراضية وتثبيت المتطلبات

```
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2) تشغيل الخادم

```
uvicorn app:app --reload
```

3) افتح المتصفح على:

```
http://127.0.0.1:8000
```



> مهم: غيّر كلمة المرور في `app.py` بعد أول تشغيل، وكذلك `SECRET_KEY`.

## المزايا
- تسجيل وتسجيل دخول مع جلسات
- مواد وأسئلة منتصف ونهائي
- إجابات فورية وتصحيح مع شرح
- لوحة تحكم ونقاط وترتيب
- اقتراح أسئلة ورسائل
- بلاغات أخطاء
- لوحة إدارة كاملة
- تصدير/استيراد CSV للأسئلة
